### What is WithCoach?

WithCoach is a coaching marketplace that connects learners with vetted local coaches. We solve the pain of finding quality instruction abroad — no more Facebook group hunting, cash-only transactions, or language barriers.

Coaches keep 100% of their rate. Students pay a transparent platform fee for booking infrastructure, payment protection, and auto-translation.

- **Frontend** — Next.js, React, Tailwind CSS
- **Backend** — Node.js, Supabase (Postgres, Auth, Storage)
- **Payments** — Stripe
- **Infrastructure** — Vercel
